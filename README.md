## 1.SQL

      SQL 是用于访问和处理数据库的标准的计算机语言
      SQL 能做什么？
      SQL 面向数据库执行查询
      SQL 可从数据库取回数据
      SQL 可在数据库中插入新的记录
      SQL 可更新数据库中的数据
      SQL 可从数据库删除记录
      SQL 可创建新数据库
      SQL 可在数据库中创建新表
      SQL 可在数据库中创建存储过程
      SQL 可在数据库中创建视图
      SQL 可以设置表、存储过程和视图的权限
